# 计算器 (Calculator)

一个使用 Python tkinter 库构建的简洁桌面计算器应用程序，具有基础算术运算和历史记录功能。

![Python Version](https://img.shields.io/badge/python-3.6+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 功能特性

- **基础运算**: 支持加法 (+)、减法 (-)、乘法 (×)、除法 (÷)
- **实用功能**:
  - 清除 (C) - 重置计算器状态
  - 正负切换 (±) - 快速切换数值正负
  - 百分比 (%) - 一键计算百分比
- **智能显示**: 超过12位数字自动转换为科学计数法显示
- **历史记录**:
  - 自动保存最近50条计算记录
  - 双击 C 按钮可快速清空历史
  - 自动滚动到最新记录
- **界面优化**: 固定窗口大小 480×760，布局清晰易用

## 运行环境

- Python 3.6 或更高版本
- 操作系统: Windows / macOS / Linux (需要图形界面支持)

## 使用方法

### 直接运行

```bash
python cu.py
```

### 界面操作

| 按钮 | 功能说明 |
|------|----------|
| 0-9  | 输入数字 |
| .    | 小数点 |
| + - × ÷ | 四则运算 |
| =    | 计算结果 |
| C    | 清除 (单击) / 清空历史 (双击) |
| ±    | 切换正负号 |
| %    | 转换为百分比 |

## 代码结构

```
cu.py
├── Calculator 类
│   ├── __init__()          # 初始化 UI 和状态
│   ├── on_button_click()   # 按钮点击路由
│   ├── input_number()      # 处理数字输入
│   ├── input_operation()   # 处理运算符
│   ├── calculate_result()  # 执行计算
│   ├── clear_all()         # 清除状态
│   ├── clear_history()     # 清空历史记录
│   ├── toggle_sign()       # 切换正负号
│   ├── percentage()        # 百分比计算
│   ├── update_display()    # 更新显示屏
│   └── run()               # 启动主循环
└── main 入口
```

### 布局架构

- **7行 × 4列** 网格布局
- 第0行: 显示屏 (固定高度)
- 第1-5行: 按钮区域 (主要空间)
- 第6行: 历史记录区 (底部)

### 状态管理

| 变量 | 说明 |
|------|------|
| `current_input` | 当前输入的数字 |
| `previous_input` | 上一个操作数 |
| `operation` | 当前运算符 |
| `should_reset_display` | 下次输入时重置显示 |
| `history` | 历史记录列表 (最多50条) |

## 注意事项

- 由于使用 tkinter GUI 库，运行时需要图形显示环境
- 在无头环境中运行可能会出现显示相关错误，属正常现象
- 除零操作会弹出错误提示并自动重置计算器
